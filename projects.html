Проекты на Raspberry Pi 5

Примеры проектов, которые можно реализовать на новом поколении Raspberry Pi с улучшенной производительностью процессора, поддержкой PCIe и расширенными мультимедийными возможностями

Ретро игровая консоль

Превратите свой Raspberry Pi в игровую консоль для классических игр. С помощью RetroPie вы сможете запускать игры с NES, SNES, PlayStation и других ретро-платформ.

Необходимые компоненты:

  • Raspberry Pi 5
  • Блок питания (5В, 5А)
  • USB геймпад или контроллер
  • microSD карта (32GB рекомендуется)
  • Корпус с активным охлаждением

Домашний NAS

Создайте собственное сетевое хранилище для резервного копирования, хранения медиафайлов и совместного доступа к данным в домашней сети.

Необходимые компоненты:

  • Raspberry Pi 5
  • Блок питания (5В, 5А)
  • microSD карта (16GB+)
  • USB или NVMe накопитель
  • Ethernet кабель
  • Корпус с охлаждением

Кластер из Raspberry Pi

Создайте свой собственный мини-суперкомпьютер, объединив несколько Raspberry Pi в вычислительный кластер. Идеально подходит для изучения распределенных вычислений и параллельной обработки данных.

Необходимые компоненты:

  • 8 x Raspberry Pi 5
  • 8 x Raspberry Pi PoE+ HAT
  • 8-портовый Gigabit PoE коммутатор
  • USB 3 to Gigabit Ethernet адаптер
  • USB 3 to SATA адаптер
  • SSD SATA накопитель
  • 8 x Ethernet кабели
  • microSD карты (16GB+)
  • Корпус для кластера

Pi-hole DNS сервер

Создайте собственный DNS сервер для блокировки рекламы на уровне сети. Pi-hole защищает все устройства в вашей домашней сети от нежелательной рекламы и трекеров.

Необходимые компоненты:

  • Raspberry Pi 5
  • Блок питания (5В, 5А)
  • microSD карта (8GB+)
  • Ethernet кабель
  • Корпус с охлаждением
